# Análisis Inteligente del Mercado Inmobiliario en Madrid

**Proyecto presentado a la II Edición de los Premios a la Reutilización de Datos Abiertos del Ayuntamiento de Madrid (2026)**

* **Autor:** Rodrigo Alonzo Peña Vigil
* **Categoría:** Estudios, investigaciones e ideas

## 🎯 Objetivo del Proyecto
Este proyecto presenta una solución tecnológica para facilitar la comprensión de la evolución del mercado inmobiliario en Madrid. A través de un cuadro de mando interactivo basado en datos abiertos, la herramienta permite explorar series temporales, analizar rentabilidad y comparar distritos de forma intuitiva.
